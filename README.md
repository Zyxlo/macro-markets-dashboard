# Macro + Markets Dashboard

A Streamlit dashboard that tracks major market assets and basic daily price movement.

## Current Features

- Asset selector
- Time-period selector
- Interactive price chart
- Latest close
- Daily change
- Daily percentage change
- Basic market interpretation

## Assets Tracked

- S&P 500
- Nasdaq
- Gold
- Crude Oil
- Bitcoin
- EUR/USD

## Tech Stack

- Python
- Streamlit
- yfinance
- Plotly
- pandas

## How to Run

```bash
streamlit run app.py
```
